
## 4. الأسئلة التقنية لي غادي يسولوك (جهز الرد)

### سؤال: "شنو الفرق بينك و بين ChatGPT؟"
> "ChatGPT عندو قاعدة بيانات عامة. حقي عندو RAG مخصص على القوانين المغربية. يعني كيجاوبك بالمادة و القانون الصحيح. و كاين demo mode للمغاربا لي ما عندهمش إيداع بنكي."

### سؤال: "كيفاش كاتضمن أن الـ AI ما كيغلطش؟"
> "عندنا 3 مراحل باش نضمنو الدقة:
> 1. RAG كيجيب المصدر من الـ law codes
> 2. الـ AI كيجاوب فقط على المعلومات الموجودة فـ الـ context
> 3. temperature = 0.3 يعني قليل الإبداع، كيلصق فـ المصدر"

### سؤال: "شنو الـ tech stack ديالك؟"
> "Python + FastAPI للـ backend. ChromaDB + Supabase للـ vector search. Groq Llama 3.3 70B للـ LLM. Tailwind CSS للـ frontend. Docker + GitHub Actions للـ DevOps."

### سؤال: "شنو المصدر ديال القوانين؟"
> "5 مدونات رسمية: القانون الجنائي (705 مادة)، ظهيرة الالتزامات والعقود (1,274)، القانون التجاري (809)، مدونة الأسرة (400)، مدونة الشغل (589). بالإضافة لـ 83 dataset من data.gov.ma و 30 قرار من محكمة النقض."

### سؤال: "واش جربتي مع المستخدمين؟"
> "ما زال ما جربناش رسمياً. لكن عندي demo mode كيخدم بلا ما يحتاج API key. و جاهز نطلقوه فـ 3 أيام."

### سؤال: "شنو الـ business plan ديالك؟"
> "Free tier (10 أسئلة/يوم) → Pro $9/شهر (غير محدود + عقود PDF) → Enterprise $499/شهر (API + WhatsApp bot). الهدف 1,000 Pro user + 10 Enterprise = $14k/شهر."

## 5. نصائح مهمة للعرض

### لا تقولش:
- "هاد غير MVP و ما زال فيو bugs" — قول "هاد النسخة الأولى و كاين تحسينات مستمرة"
- "ما عنديش فلوس" — قول "بنيتو بأقل التكاليف و دابا نحتاج نوسّع"
- "هاد الشي صعيب" — قول "التحدي التقني لي حليتو هو..."

### الأخطار لي تتفاداهم:
1. **لا تفتح tabs كتيرين فالمتصفح** — فقط الـ tabs الضرورية
2. **لا تنسى الـ notifications ديال Windows** — قفلهم
3. **لا تبقى صامت فاش كاتحرق الـ slides** — كمل الكلام
4. **إذا طاحت الـ server** — عندك backup: افتح الـ pitch deck من `/pitch` و كمّل العرض من غير demo

### أهم 3 نقاط تبني عليهم كل الكلام:
1. **Darija-first** — ما عندك منافس غيره
2. **0 تكلفة** — بنيتو بلا فلوس، دابا نحتاج نكبر
3. **بيانات حقيقية** — 3,777 مادة من القوانين الرسمية، مش مجرد AI كيهبد
