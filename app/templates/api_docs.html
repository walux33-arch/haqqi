🌐 API العمومي لحقي

Public API — Documentation Technique pour l'Intégration Ministérielle

🚀 نظرة عامة

API REST تمكن الوزارات والمؤسسات العمومية من دمج الاستشارة القانونية الذكية في بواباتها ومنصاتها الرقمية.

BASE_URL: https://haqqi.ma/api

Authentication: Bearer Token (JWT)

POST /chat

استشارة قانونية بالدارجة/العربية مع RAG والاجتهاد القضائي

Request Body:

{ "question": "شنو هي شروط الزواج فالمغرب؟", "session_id": "abc123", "history": [] }

Response (Streaming):

text/plain — تيار نصي بالدارجة مع المصادر القانونية
GET /stats

إحصائيات عامة: عدد المواد القانونية، الأحكام القضائية، والمصادر

GET /judgements?search=...

البحث في قرارات محكمة النقض — تصفية حسب الغرفة والسنة

POST /contracts/generate

توليد عقود PDF — عقود الكراء، الشركات، العمل...

POST /pleadings/generate

توليد المقالات القانونية (عرائض، مذكرات دفاع، استئناف...)

POST /ingestion/url

استيراد النصوص القانونية من روابط عمومية (للاستخدام الوزاري)

⚙️ حدود الاستخدام (Rate Limits)

  • المستوى المجاني: 20 طلب/دقيقة
  • المستوى الوزاري: غير محدود (بعد الاتفاقية)
  • الاشتراك: اشتراك شهري أو سنوي حسب حجم الاستخدام
